Data Conclusions

The aim of this project was to explore Lincolnshire crime data betweek the period of 01/01/2021 to 12/12/2021 from the start of the project, We can go ahead and say that the data has been explored using a series of different machine learning algorithms and python tools which further infromation can be found in the supporting document of this artifact.

Dataset Limitations

Some data may have to been reported preciecly when the crime occured leading to to anomalys in some of the top 3 tredning crimes over the year. Other limitations that where faces within the dataset was that there was not that much data made available from archives leading to the machine learning algorithms not being able to manipulate much data to produce better predictions.

Data.Police Dataset

At the time of writing the Lincolnshire Data set can be found at Data.Poilce downloads. when opened, an option to select different counties is given with drop downs of the time frame. The features within the data are as folllows.

  • Crime ID: a unique identifyer for crime records, this feature got dropped as there was no use for this during our analysis
  • Month: This features was used to produce the reports on the different crime types throughout the months of a year
  • Reported by / falls within: only signlas what polce departmented documented the crime
  • Longitude / latitude: These features was extracted into a geojson file in which helped build and create a script to map out crime points.
  • Location: Was used to calculate where the most crimes are comitted
  • LSOA Code / Name: These features where used to extract data that belonged to just Lincoln city for the map.
  • Crime type: This feature was used to extract exactly what crime type was comitted
  • Last Outcome: Used to identify what happened with the case
  • Context: Had no information due to security reassons. Was dropped from dataframe

Conclusions

Main points to be discussed within the data analysis of the crime are:

  • 23431 Violence and secual offences occured in the year 23431, criminal damage offences accounted for 6010 if the total crime and, public order offences where at 4876, These where the top 3 crimes
  • Most of the top 3 crimes in Lincolnshire, esepcially thoose that was violent and sexual lead to a suspect not being procecuted. (This was over 14000 of the cases
  • Most of the crimes in Lincoln appear to happen at a supermarket or a parking area, with the least amount being committed near educational buildings
  • The random forest algorithm was the best at predicting target crime types with the hhighest predicted crime type being voiolence and sexual offenses being the most accurate with 0.92 recall and 0.70 f1-score.
  • Judging by the map implemented within the website, it appears that most cases of crime occur within the city center and not around educational institutions.
