The aim of this project is to analyse Lincolnshire crime data using a series of different python tools while also using machine learning to measure the outcome of crime predictions

Overview of Lincolnshire crime

The map below shows the locations of crime occurence in the form of a cluster map per crime reported cumulative (The total number of cases in that region so far) figures.

The data is for the dates 01/01/2021 until 31/12/2021 inclusive.

Scroll in and out to view different layers of the cluster map and click on data point to reveal recent crime occurences.
